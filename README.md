# Explainable Machine Learning for Colorectal Cancer Prediction Using Gut Microbiome Data

## Project Overview

This project aims to develop an explainable machine learning pipeline for predicting colorectal cancer using gut microbiome taxonomic profiles derived from shotgun metagenomic sequencing.

## Objectives

- Explore microbiome composition
- Compare machine learning models
- Interpret predictions using SHAP
- Identify microbial biomarkers associated with colorectal cancer

## Tools

- Python
- Google Colab
- Pandas
- Scikit-learn
- XGBoost
- SHAP
- Matplotlib
- Plotly
